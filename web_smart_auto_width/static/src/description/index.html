Smart List Auto Width

Automatically adjust Odoo list column widths so that headers and content are fully visible.

Main Features

  • 📏 Auto-calculates column width to avoid truncation
  • 📝 Header-first logic: if the header is wider than content, the header width is preserved
  • 📊 Content check: scans all cells to ensure the longest value is visible
  • ⚡ Works globally: applies to all list views and one2many/many2many tables in forms
  • 🎨 Fully compatible with Odoo’s native column compression algorithm
  • 🔧 Zero configuration: works immediately after installation

Installation

  • Place the module in your Odoo addons path
  • Update the apps list and install Smart List Auto Width
  • Refresh your browser cache (Ctrl + F5) to activate
